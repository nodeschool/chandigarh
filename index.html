Node.js

Node.js was created in 2009 by Ryan Dahl and different engineers working at Joyent. Node.js was initially released in 2009 supporting just Linux. Its advancement and maintenance was driven by Dahl and supported by Joyent, the firm where Dahl worked. Node.js is an open-source, cross-stage runtime environment for creating server-side web applications.

Node.js allows the formation of web servers and networking tools utilizing JavaScript and a gathering of "modules" that handle different cor functionality. Modules handle file system I/O, networking, binary information, cryptography functions, data streams[34] and other center functions.
